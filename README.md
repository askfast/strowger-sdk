# Strowger Java SDK

The SDK to interact with the strowger API.

##Installation

```
<dependency>
  <groupId>com.askfast.strowger.sdk</groupId>
  <artifactId>strowger-sdk</artifactId>
  <version>0.1.0</version>
</dependency>
```
